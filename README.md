# CodeAlpha Image Gallery

## Overview
This project is a simple and interactive Image Gallery developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

The gallery allows users to browse images using Previous and Next buttons with a clean and responsive user interface.

## Features
- Display multiple images
- Previous and Next navigation buttons
- Responsive design for different screen sizes
- Hover zoom effect on images
- Simple and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

```
CodeAlpha_ImageGallery
│
├── index.html
├── style.css
├── script.js
└── images
    ├── 1.jpg
    ├── 2.jpg
    ├── 3.jpg
    └── 4.jpg
```

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server.
4. Click the Previous and Next buttons to navigate through the images.
